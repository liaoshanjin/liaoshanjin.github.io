Centos7 安装 GOGS

24 Oct 2019
@特污兔

环境 默认 已经在 centos7 上安装了 LNMP1.5

准备

添加一个 git 用户

useradd git 

passwd git

切换到 git 用户

su git

下载 安装 gogs 二进制安装包

地址摸我

cd ~

wget https://dl.gogs.io/0.11.86/gogs_0.11.86_linux_amd64.tar.gz

tar -zxvf gogs_0.11.86_linux_amd64.tar.gz

cd gogs

./gogs web

然后执行安装

守护进程

虽然我们的 gogs 安装成功了，但是一关闭的话就不能访问了。

复制 gogs.service 文件

cp /home/git/gogs/scripts/systemd/gogs.service    /usr/lib/systemd/system/

systemctl enable gogs.service

systemctl start gogs.service

Nginx 设置反向代理

因为 /usr/local/nginx/conf/nginx.conf 已经引入了 vhost 下的所有 conf 文件

所有我们只需要在 /usr/local/nginx/conf/vhost/ 下面新建一个 gogs.conf 即可：

cd /usr/local/nginx/conf/vhost/

vim gogs.conf

添加一下内容，保存退出

server {
    listen 80;
    server_name gogs.test;

    location / {
        proxy_pass http://127.0.0.1:3000;
    }
}

重启 NGINX 。

lnmp nginx restart

便可在浏览器中输入 gogs.test 访问 gogs。
