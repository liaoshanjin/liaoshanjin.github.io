安装完 Homestead 后的 C 盘太满

24 Oct 2019
@苦味糖

问题

这几天才发现自己的 C 盘一直太满了，都见红了。直到清完了一部分大文件才终于变蓝了。

即便如此，还是可以看到：

可以看到系统盘太满了，在具体点,我们进入 C:\Users\Administrator 目录，查看 VirtualBox VMs 的大小：

这还只是我刚装完 Homestead 的时候，多点项目，随便上十几个G。作为强迫症患者，决定把他移到别的盘。

方法

第一步

把 C:\Users\Administrator 下的 VirtualBox VMs 和 .VirtualBox 两个文件夹拷贝到 E:\VirtualBox 下。

第二部

以管理员的身份打开 VirtualBox 软件 ，选择左上角的 管理 》 全局设定 看到如下页面：

在 默认虚拟电脑位置 选项里面设置我们刚刚 有的 E:\VirtualBox\VirtualBox VMs 文件夹。

重启 Homestead 查看效果：

vagrant.exe reload --provison

总结

以上只是针对已经安装的 homestead, 其实背后我也是第一次弄。远比这麻烦。最好的办法还是在安装 homestead 的时候在 未启动 homestead 之前执行第二部。
