Hugo 常用命令

hugo server -D  -t theme_name -b url_name

-D 等价于 --buildDrafts
-t 主题名称(theme_name)
-b 基础路由(url_name)

基于以上，如果我们想在本地已经搭建有 hugo 环境，做一下操作：

启动 hugo 服务，浏览本地的博客：

hugo server -D -t theme_name

hugo server -D -t hugo-coder

# 启动当前主题
hugo server -D -t yinyang

在添加问博客，想生成静态文件，上传到 github :

hugo --theme=hugo-coder --baseUrl="https://liaoshanjin.github.io" --buildDrafts


hugo --theme=yinyang -b="https://liaoshanjin.github.io" -D
