PHPstorm 的一些笔记

24 Oct 2019
@特污兔

开发这些年，用过的 IDE 可能算多的了。从最早的 HBuilder 到 Sublime ,在到 ATom , VScode , Eclipse For PHP 到最后的 PHPstorm 。只有 PHPstorm 让我不会有在更改的可能。所以下面我就记一下个人的 PHPstorm 的设置。

主题

主题使用的是 Material Theme Ui ,很喜欢这个主题。

快捷键系列

  • ctrl + shift + A 快速搜索配置项
    • 例如去掉面包屑 ctrl + shift + A ,弹出搜索框后输入 breadcrumb ,然后在选择 don't show
    • 例如去掉折叠线ctrl + shift + A,弹出搜索框后输入code folding,然后去掉显示代码折叠大纲的勾，保存
    • 去掉参数提示ctrl + shift + A,弹出搜索框后输入show parameter name hits,然后去掉show parameter name hits的勾
    • 去掉右边的竖线Show right margin
  • ctrl + shift + N 快速打文件,弹出搜索框后输入你想打开的文件
  • ctrl + F12 查看类的方法（文件结构）, 弹出的文件结构可以直接搜索
  • ctrl + E 打开最近编辑或打开的文件列表, 弹出的文件结构可以直接搜索
  • ctrl + alt + shift + N 方法搜索，能快速定位到方法
  • ctrl + alt + L 快速的排版
  • shift + F6 选择要统一修改的内容，然后可以统一编辑
  • ctrl + ~ 快速打开主题选择
  • ctrl + j 启动 live template
  • ctrl + z 撤销
  • ctrl + shift + z 反撤销

自定义快捷键

  • ctrl + shift + A,弹出搜索框后输入keymap, 然后点击右边的搜索框,按你目前的快捷键,然后可以设置

自定义类的注释头

  • ctrl + shift + A,弹出搜索框后输入file template, 点击模板,选择PHP FILE,因为他有包含PHP File Header.php,点击包含切换, 选择PHP File Header进行编辑,保存

自定义排版样式

  • ctrl + shift + A,弹出搜索框后输入code style, 选择php

自定义主题

  • 打开主题网站
  • 选择要换的主题，点击codepen.icls
  • 复制全部内容
  • 打开目录C:\Users\Administrator\.PhpStorm2018.1\config\colors,新建一个.icls的文件，如codopen.icls
  • 将复制的内容黏贴就好。然后选择这个主题

解决插件下载慢的问题

  • ctrl + shift + A, 弹出搜索框后输入updates, Use secure connection 的 勾去掉就可以了,

选中高亮

  • Ctrl + Alt + S 打开安装插件, 并安装插件 BrowseWordAtCaret
  • 在 Settings - Editor - General - Appearance - Browse Word At Caret 下勾选设置
  • 在 Settings - Editor - Color Scheme - BrowerWordAtCaret 页面设置颜色

设置支持 TODO、FIXME、XXX 等特殊设置

  • Ctrl + Shift +A 输入 TODO ,选择 setting, 点击右上角的添加, pattern 输入 \bxxx\b.*, 勾选 use coloer themes ... 即可

快捷设置系列

此系列基于 Shift + Ctrl +A 呼出设置搜索

  • 勾选掉不需要的插件提示

    live templadtes

  • 个人代码风格

    code style, 弹出设置后  选择 PHP, 点击 set from, 然后选择 WordPress

  • 设置等号对齐

    code style, 弹出设置后  选择 PHP, 点击导航 Wrapping and Braces 最后勾选
Align consecutive assignments 和 Align key-value pairs

  • 快捷键创建新文件

    Alt + Home 键可以定位你需要创建文件的目录， 然后 Alt + Insert 键新建你想要创建的文件类型

  • 重构代码

    Ctrl + Shift + Alt + T ,弹出的对话框中选择 Methons , 自选重构

  • 多点编辑

    鼠标移动到需要编辑单词的后面, 使用快捷键 Alt + J ，随后可多点编辑

设置 Terminal 的字体

打开设置 Editor >> Code Scheme >> Console Font 选择想要设置的字体。

我的字体是 Laravel 作者的字体 Operator . 下载地址[摸我](http://148.70.84.101/storage/resources//Operator Mono.zip)
