一. 将秒数转换成时分秒格式

$time = 900; //秒
return (new \DateTime('@0')) ->diff(new \DateTime("@$time")) ->format('%H:%I:%S');

二. Linux 修改主机名

三. 快速获取 PHP 配置文件位置

Scan for additional .ini files in: /usr/local/php/conf.d

四. 快速生成6位由数字字母组成的字符串

function rand_str()
{
    static $max;
    $max = $max ? $max : \base_convert('zzzzzz', 36, 10);
    try {
        return \strtoupper(\str_pad(\base_convert(\random_int(0, $max), 10, 36), 6, '0', STR_PAD_LEFT));
    } catch (\Exception $e) {
        return false;
    }
}

五. 从GitHub克隆加速

从 GitHub clone 一个项目有时候非常慢。我们可以借用 GitClone :

# 比如我要克隆 https://github.com/koel/koel  这个项目
git clone https://gitclone.com/github.com/koel/koel.git

# 格式：
git clone https://gitclone.com/github.com/作者/项目
